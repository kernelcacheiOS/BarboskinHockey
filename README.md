# Хоккейная Арена - Мобильная Игра

Адаптивная хоккейная арена, созданная на HTML/CSS/JavaScript для мобильной игры с использованием оригинального изображения арены.

## Особенности

- **Использует оригинальное изображение арены** - импортированное изображение arena.jpg
- **Полностью адаптивный дизайн** - отлично работает на всех устройствах
- **Интерактивные зоны** - клики по кругам вбрасывания и центральному кругу
- **Автоматический поворот** - арена поворачивается на мобильных устройствах в портретной ориентации
- **Touch-оптимизация** - поддержка сенсорных экранов
- **Анимации и эффекты** - плавные переходы и визуальная обратная связь
- **PWA готовность** - можно установить как приложение на мобильные

## Структура проекта

```
├── index.html          # Основная HTML структура
├── styles.css          # Адаптивные стили
├── script.js           # Интерактивность и анимации
├── manifest.json       # PWA конфигурация
├── resources/
│   └── arena.jpg       # Оригинальное изображение арены
└── README.md           # Документация
```

## Адаптивность

### Мобильные устройства (до 480px)
- Уменьшенные элементы для лучшей видимости
- Автоматический поворот в портретной ориентации
- Оптимизированные размеры кругов и звезд

### Планшеты (481px - 768px)
- Средние размеры элементов
- Сохранение пропорций арены

### Планшеты альбомные (769px - 1024px)
- Увеличенные элементы
- Лучшая детализация

### Десктоп (1025px+)
- Максимальные размеры
- Полная интерактивность с эффектами наведения

## Запуск

Просто откройте `index.html` в любом современном браузере.

## Технические детали

- Использует CSS Grid и Flexbox для адаптивной верстки
- CSS Custom Properties для легкой настройки цветов
- Intersection Observer API для оптимизации анимаций
- Touch events для мобильных устройств
- Media queries для всех популярных разрешений
